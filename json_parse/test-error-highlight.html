错误高亮功能测试

测试用例1：缺失引号的属性名

{
  name: "测试",
  age: 25
}

预期：第2行应该被高亮显示

测试用例2：单引号字符串

{
  "name": '测试',
  "data": 'value'
}

预期：第2、3行应该被高亮显示

测试用例3：尾随逗号

{
  "name": "测试",
  "age": 25,
}

预期：第3行应该被高亮显示

测试用例4：混合错误

{
  name: '测试',
  age: 25,
  data: [1,2,3,],
  active: true
}

预期：第2、3、4行应该被高亮显示

测试说明：将上述JSON字符串复制到JSON解析工具中，查看错误行是否被正确高亮显示。
