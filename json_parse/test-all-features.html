JSON解析工具 - 所有功能测试

1. 满屏展示和区域调节

测试步骤：

  • 打开JSON解析工具页面
  • 确认页面占据整个浏览器窗口
  • 拖动左右区域之间的分隔线
  • 验证区域大小可以调节

预期：页面满屏显示，区域大小可手动调节

2. 行号显示功能

测试步骤：

  • 在左侧输入框中输入多行JSON
  • 查看左侧是否显示行号
  • 滚动输入框，验证行号同步滚动
  • 查看右侧结果区域是否显示行数统计

预期：左右区域都显示行号/行数统计

3. 双击复制功能

测试步骤：

  • 解析一个有效的JSON
  • 在右侧结果区域双击
  • 粘贴到文本编辑器验证内容

预期：双击后内容被复制到剪贴板

4. 压缩选项

测试输入：

{ "name": "测试", "age": 25, "data": [1, 2, 3] }

测试步骤：

  • 勾选"压缩"选项
  • 查看结果是否压缩为一行
  • 取消勾选"压缩"选项
  • 查看结果是否格式化显示

预期：压缩选项控制JSON输出格式

5. 着色选项

测试输入：

{ "string": "测试文本", "number": 123.45, "boolean": true, "null": null, "array": [1, 2, 3] }

测试步骤：

  • 勾选"着色"选项
  • 查看结果中不同类型是否有不同颜色
  • 取消勾选"着色"选项
  • 查看结果是否变为单色

预期：着色选项控制语法高亮显示

6. 显示JS类型

测试输入：

{ "string": "文本", "number": 42, "boolean": false, "null": null, "object": {"key": "value"}, "array": [1, 2, 3] }

测试步骤：

  • 勾选"显示JS类型"选项
  • 鼠标悬停在各个值上查看提示
  • 验证类型显示是否正确

预期：鼠标悬停时显示JavaScript类型

7. 显示数组索引

测试输入：

{ "items": ["苹果", "香蕉", "橙子"], "numbers": [10, 20, 30, 40] }

测试步骤：

  • 勾选"显示数组索引"选项
  • 查看数组元素前是否显示索引
  • 验证索引显示是否正确

预期：数组元素前显示索引号

8. 错误高亮功能

测试输入：

{ name: '测试', age: 25, data: [1,2,3,], active: true }

测试步骤：

  • 输入包含错误的JSON
  • 查看左侧行号区域是否有红色高亮
  • 验证错误行号是否正确

预期：错误行号以红色背景高亮显示

9. 综合测试

测试步骤：

  • 同时启用所有选项
  • 输入复杂的JSON数据
  • 验证所有功能协同工作正常
  • 测试选项切换时的响应速度

预期：所有功能协同工作，无冲突
