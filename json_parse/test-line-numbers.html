问题：行数展示不对，只展示了1行
原问题：无论输入多少行内容，行号区域只显示1行
修复方案：
  • 合并重复的input事件监听器
  • 确保updateLineNumbers()在每次输入时都被调用
  • 移除HTML中硬编码的行号
  • 在应用初始化时调用updateLineNumbers()

{"name": "test", "value": 123}

测试用例3：空输入

核心逻辑：
// 输入变化时同时更新行号和解析JSON
this.jsonInput.addEventListener('input', () => {
    this.updateLineNumbers(); // 实时更新行号
    this.debounce(() => this.parseJSON(), 300); // 防抖解析
});
